# Player
适配群晖Audio StationAPI自实现播放器APP。利用空闲时间自己开发满足自己使用，独乐乐不如众乐乐，分享出来大家用。由于个人能力有限并且时间不是很充裕，有什么崩溃的请谅解，我会尽力解决和持续优化！
